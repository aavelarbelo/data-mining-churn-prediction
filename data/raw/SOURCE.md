# Data Source

**Dataset:** Telco Customer Churn public sample dataset.

**Origin:** Publicly available customer churn dataset commonly used for educational and portfolio projects.

**Usage:** Educational and portfolio purposes only.

**Target variable:** `Churn` (Yes/No).

## Description

This dataset contains customer information from a telecom context, including demographic data, contracted services, account information, charges and churn status.

The goal of this project is to use the dataset to build a supervised machine learning workflow for customer churn prediction.

## Column Groups

The main column groups are described in the main `README.md` file:

- Demographic information
- Contracted services
- Contract and billing information
- Financial values
- Target variable

## Notes

This dataset is used only for learning, experimentation and portfolio demonstration.

The project does not claim production-level validity and should not be interpreted as a real business deployment without further validation, monitoring and governance.
